# Amazon Books Scraper (Scrapy)

This project is a **Scrapy web scraper** that extracts book information from **Amazon India search results**.  
The spider collects details such as **book name, author, and price** and supports **pagination** to scrape multiple pages.

---

## Features

- Scrapes Amazon book listings
- Extracts:
  - Product Name
  - Author
  - Price
- Supports scraping multiple pages using pagination
- Built using **Python Scrapy framework**
- Data can be exported to **JSON, CSV, or XML**
